# Dotfiles

## 1: Dotfile 备份管理

* dotfile 文件备份管理, 备份 ideavim nvim zsh 等常用软件的配置文件

## 2: Dotfile 目录结构

```
~/.config/dotfiles
|------ nvim
|    |--- init.vim
|	 |--- lua
|	   |---basic.lua
|	   |---init.lua
|	   |---kedbindings.lua
|	   |---plugins.lua
|	   |---plugin-config
|	 |--- plugin
|------- ideavim
```
